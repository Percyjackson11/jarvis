Certainly! Here's a step-by-step guide on how to control two bulbs using a relay module and Arduino through Python:

Hardware Setup:
1. Arduino board (e.g., Arduino Uno)
2. 2-channel relay module
3. Two bulbs with appropriate sockets and power supply
4. Jumper wires

Connections:
1. Connect the VCC and GND pins of the relay module to 5V and GND on the Arduino.
2. Connect the IN1 and IN2 pins of the relay module to digital pins 7 and 8 on the Arduino.
3. Connect each bulb to the normally open (NO) terminal and common (COM) terminal of each relay channel.
4. Connect the power supply to the bulbs and relay module as per their specifications.

Arduino Code:
Upload the following code to your Arduino:

```cpp
int relay1 = 7;
int relay2 = 8;

void setup() {
  pinMode(relay1, OUTPUT);
  pinMode(relay2, OUTPUT);
  digitalWrite(relay1, HIGH);
  digitalWrite(relay2, HIGH);
  Serial.begin(9600);
}

void loop() {
  if (Serial.available() > 0) {
    char command = Serial.read();
    switch (command) {
      case '1':
        digitalWrite(relay1, LOW);
        break;
      case '2':
        digitalWrite(relay1, HIGH);
        break;
      case '3':
        digitalWrite(relay2, LOW);
        break;
      case '4':
        digitalWrite(relay2, HIGH);
        break;
    }
  }
}
```

Python Code:
Now, you can use the following Python script to control the bulbs:

```python
import serial
import time

# Establish serial connection
ser = serial.Serial('COM3', 9600)  # Replace 'COM3' with your Arduino's port
time.sleep(2)  # Wait for the connection to establish

def control_bulb(bulb_number, state):
    if bulb_number == 1:
        command = '1' if state == 'on' else '2'
    elif bulb_number == 2:
        command = '3' if state == 'on' else '4'
    else:
        print("Invalid bulb number")
        return
    
    ser.write(command.encode())
    print(f"Bulb {bulb_number} turned {state}")

# Example usage
try:
    while True:
        bulb = int(input("Enter bulb number (1 or 2): "))
        state = input("Enter state (on/off): ").lower()
        
        if state not in ['on', 'off']:
            print("Invalid state. Use 'on' or 'off'.")
            continue
        
        control_bulb(bulb, state)

except KeyboardInterrupt:
    print("Program terminated")
    ser.close()
```

How it works:
1. The Arduino code listens for commands via the serial port.
2. The Python script establishes a serial connection with the Arduino.
3. The `control_bulb` function sends the appropriate command to the Arduino based on the bulb number and desired state.
4. The Arduino receives the command and switches the corresponding relay on or off.

To use the script:
1. Upload the Arduino code to your board.
2. Run the Python script.
3. Enter the bulb number (1 or 2) and the desired state (on or off) when prompted.

Make sure to install the `pyserial` library (`pip install pyserial`) before running the Python script. Also, replace 'COM3' in the Python code with the correct port for your Arduino.

This setup allows you to control two bulbs independently using Python commands through the Arduino and relay module.
